# IESN train

This web app is designed to create some quizzes like Quizlet. But for IESN folklore training.

## Features

- Landing page list of quizzes
- Quiz taking interface (multiple Layouts based on question type)
- Last quiz should be saved in local storage

## Run (dev)

Start the local dev server:

```bash
npm install
npm run dev
```

The app uses Vite + Tailwind. Tailwind config is in `tailwind.config.cjs` and styles are imported from `src/main.css`.
